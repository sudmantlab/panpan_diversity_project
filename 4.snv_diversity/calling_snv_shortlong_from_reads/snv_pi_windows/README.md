# windowed π comparison (long vs short reads)


> *All comparisons use 10 kb non-overlapping windows of nucleotide diversity
> (π) computed with **pixy** from all-sites VCFs of long-read- and
> short-read-derived calls aligned to mPanTro3. Hybrid and NC populations
> were excluded; analyses are restricted to autosomes. Region overlap:
> Centromere/Telomere from `mPanTro3_genomefeatures.bed`; CenSat from
> `mPanTro3_CenSat.bed`; SegDup from `mPanTro3_sedefSegDups.bed`; TR from
> `pantro_catalog.no_overlaps_simp.bed`; SRA from `mPanTro3_SR_mask.bed`.
> A window is assigned to the highest-priority region it overlaps
> (Cen > Telo > CenSat > SegDup > TR > Other).*
